Passo 1: Preparação

Certifique-se de que você possui o Minecraft Java Edition instalado no seu computador.
Localize o arquivo zip do mundo chamado "creations" que você deseja instalar. Certifique-se de que ele está salvo em um local de fácil acesso.
Passo 2: Descompactação

Crie uma pasta em algum lugar do seu computador onde você deseja armazenar os mundos customizados. Pode ser uma pasta chamada "Mundos Customizados" ou algo similar.
Mova o arquivo zip "w creations" para a pasta que você criou.
Clique com o botão direito no arquivo zip e escolha a opção "Extrair aqui" ou "Descompactar aqui". Isso irá extrair os arquivos do mundo para a pasta onde o arquivo zip está localizado.
Passo 3: Mover para a Pasta de Mundos do Minecraft

Abra o menu Iniciar e digite "%appdata%" (sem aspas) e pressione Enter. Isso abrirá a pasta "Roaming" no seu computador.
Dentro da pasta "Roaming", você verá uma pasta chamada ".minecraft". Abra essa pasta.
Dentro da pasta ".minecraft", procure por uma pasta chamada "saves". Esta é a pasta onde você deve colocar os mundos personalizados.
Arraste a pasta do mundo "creations" que você extraiu anteriormente para a pasta "saves". Agora o mundo estará disponível para jogar.
